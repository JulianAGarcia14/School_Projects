	It's better to use a structure array when you're filling the array with more than one type of value. With a structure array you can assign more than one value to each index of the array. This is helpful for when a program requires a list with multiple attributes.
	It's better to use an array when you have a simple list with one value per index value. It's better simply because using a structure would be an unnecessarily long way to store those values. With structures you have to go through the effort of declaring the structure and using the structure notation for whenever the array is needed, while with regular arrays there's nothing more to write than the array itself.
